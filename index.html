About Me

Hi, I’m Myron Holmes—a software developer based in Charleston, SC.

My journey into tech began in 2023 at Devmountain, where I built my first two full-stack projects: one with vanilla JavaScript and another with Java. Since then, I’ve expanded my skills into modern frameworks like Svelte, React, and Python, while deepening my backend expertise.

I enjoy bringing ideas to life through clean, efficient code and intuitive user experiences. My toolkit spans Java, TypeScript, Spring Boot, SvelteKit, Angular, PostgreSQL, and more.
